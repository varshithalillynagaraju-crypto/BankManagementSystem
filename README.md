# Bank Management System

## Overview
This project is a simple Bank Management System developed using Java and Object-Oriented Programming (OOP) concepts. It simulates basic banking operations such as deposits, withdrawals, balance checking, and transaction history tracking.

## Features
- Create a bank account with an initial balance
- Deposit money
- Withdraw money
- Check account balance
- View transaction history
- Menu-driven console interface

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## Project Structure
- Account Class
  - Deposit Method
  - Withdraw Method
  - Balance Management
  - Transaction History

- BankManagementSystem Class
  - User Interaction
  - Menu-Driven Operations

## How to Run

1. Save the file as `BankManagementSystem.java`
2. Open terminal or command prompt
3. Compile the program
